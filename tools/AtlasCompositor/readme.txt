Использование PngToAtlas из командной строки

Ключи:
-a Собрать текстуры в один атлас
-c ATF-компрессия
-j JPEG-компрессия с PNG-альфой
-b Добавить бинарное представление прозрачности
-s Объединить в атлас как sprite sheet
-w Обрезать прозрачность вокруг изображения
-e Продублировать крайние пиксели изображения

Пример использования:

PngToAtlas.exe -ajw "C:\Images\test1.png" "C:\Images\test2\"

Объединить в один атлас с JPEG-сжатием и обрезкой прозрачности test1.png и все картинки из папки test2\
